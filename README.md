## Repo information
This repository contains all the practical work completed for the CP1404 course at JCU. Each practical is organized into its own folder, named according to the practical number (e.g., Practical_01, Practical_02, etc.).

## Clean Code Practices

- Meaningful Names in variables functions and classes
- Consistent Indentation and Spacing
- Functions should do one thing
- Avoid Magic Numbers
- Commenting and Documentation
- Error Handling
